# CamFTP V2

Universal camera snapshot uploader for PHP.

CamFTP permet de récupérer des snapshots depuis des caméras IP et de les transférer automatiquement vers des serveurs FTP, FTPS ou SFTP.

Projet compatible avec l'utilisation historique Eedomus de CamFTP V1.

---

## Fonctionnalités

- Capture de snapshots depuis des caméras IP HTTP
- Transfert vers :
  - FTP
  - FTPS
  - SFTP
- Support multi-caméras
- Support multi-serveurs
- Mode capture globale (`numcam=99`)
- Nombre de captures configurable
- Délai entre captures configurable
- Configuration externe
- Validation stricte des paramètres
- Compatible PHP 8.1+
