# Changelog

## 2.0.0

Nouvelle version majeure.

### Ajouts

- Configuration externe via config.php
- Support FTP
- Support FTPS
- Support SFTP
- Gestion multi-caméras
- Validation stricte des paramètres GET
- Téléchargement unique des snapshots
- Gestion centralisée des erreurs
- Nommage des fichiers configurable

### Changements

- Suppression de l'envoi mail
- Suppression des variables statiques cammax / ftpmax
- Réorganisation complète du code
- Compatible PHP 8.1+

### Compatibilité

Les appels historiques CamFTP V1 restent fonctionnels.
